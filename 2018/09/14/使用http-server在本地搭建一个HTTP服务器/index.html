使用http-server在本地搭建一个HTTP服务器

发表于 2018-09-14 | 分类于 安装与配置

目的

为了方便在其他设备上访问PC上的资源，比如查看一下写好的网页在移动设备上的显示效果；在局域网的多机器中共享文件等等。在本地搭建一个HTTP服务器。这样其他设备只需要一个地址就能访问PC上的资源
